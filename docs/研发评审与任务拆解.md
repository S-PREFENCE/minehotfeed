## 三、API 接口设计

### 3.1 后端 API

```
GET /api/v1/hotspots
```

**Query Parameters:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `region` | string | 否 | `domestic` / `international` / `all`，默认 `all` |
| `category` | string | 否 | 分类筛选，默认全部 |
| `platforms` | string | 否 | 逗号分隔的平台名，如 `weibo,douyin` |
| `keyword` | string | 否 | 搜索关键词 |
| `limit` | int | 否 | 返回数量，默认 50 |
| `offset` | int | 否 | 分页偏移 |

**Response:**

### 3.2 前端数据流

```
用户进入页面
    │
    ├──► GET /api/v1/hotspots ──► 渲染页面
    │
    ├──► 每 10 分钟自动轮询 /api/v1/hotspots
```

## 四、数据库设计

### 4.1 表结构

```sql
-- 原始热点数据表（每个平台每条原始数据）
CREATE TABLE raw_hotspots (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    platform VARCHAR(32) NOT NULL,        -- 平台名
    title VARCHAR(500) NOT NULL,           -- 原始标题
    original_heat BIGINT,                  -- 原始热度值
    rank INTEGER,                          -- 排名
    url VARCHAR(2048),                     -- 原始链接
    tags JSON,                             -- 标签 ["新","爆"]
    thumbnail VARCHAR(2048),               -- 缩略图
    region VARCHAR(16) DEFAULT 'domestic', -- domestic/international
    fetched_at DATETIME NOT NULL,          -- 抓取时间
    batch_id VARCHAR(64) NOT NULL          -- 批次ID（同一次抓取共享）
);

-- 去重后的热点表
CREATE TABLE dedup_hotspots (
    id VARCHAR(64) PRIMARY KEY,            -- dedup_xxx
    title VARCHAR(500) NOT NULL,           -- 去重后标题
    unified_heat FLOAT,                    -- 统一热度分 0-100
    category VARCHAR(32),                  -- 分类
    trend VARCHAR(16),                     -- rising/stable/falling
    primary_source_id INTEGER REFERENCES raw_hotspots(id),
    secondary_source_ids JSON,             -- [2, 5, 8]
    region VARCHAR(16),
    created_at DATETIME NOT NULL,
    updated_at DATETIME NOT NULL
);

-- 索引
CREATE INDEX idx_raw_platform ON raw_hotspots(platform, fetched_at);
CREATE INDEX idx_raw_batch ON raw_hotspots(batch_id);
CREATE INDEX idx_dedup_region ON dedup_hotspots(region, updated_at);
CREATE INDEX idx_dedup_category ON dedup_hotspots(category);
```

### 4.2 数据清理策略

```
每 5 天执行一次清理任务：
  DELETE FROM raw_hotspots WHERE fetched_at < datetime('now', '-5 days');
  DELETE FROM dedup_hotspots WHERE updated_at < datetime('now', '-5 days');
  VACUUM;
```

## 七、设计验收点

### 7.1 视觉一致性

- [ ] 色彩：亮色模式白色底 + 蓝色主色调；暗色模式深灰底
- [ ] 字体：系统默认字体栈（-apple-system, ...）
- [ ] 间距：8px 基础网格
- [ ] 圆角：卡片 12px，按钮 8px
- [ ] 阴影：浅阴影（卡片悬浮效果）

### 7.2 交互一致性

- [ ] 点击热点卡片 → 新标签页打开原文
- [ ] 筛选标签 → 即时切换，无页面刷新
- [ ] 搜索 → 输入 300ms 防抖后请求
- [ ] 暗色切换 → 0.3s 过渡动画
- [ ] 数据刷新 → 顶部状态栏更新"X分钟前"
